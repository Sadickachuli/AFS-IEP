# Brand Assets

This directory contains core brand assets:

1. `logo.png` - Main logo with transparent background
   - Used in website navigation and light backgrounds
   - Recommended size: At least 512x512px for high-quality scaling
   - Format: PNG with transparency
   - Location: `/images/logo.png`

2. `logo-white.png` - White version of the logo
   - Used on dark backgrounds (e.g., footer)
   - Same dimensions as main logo
   - Format: PNG with transparency
   - Location: `/images/logo-white.png`

Logo Requirements:
- Must have transparent background
- Should be clearly visible at both large and small sizes
- Should maintain good contrast in both light and dark modes
- Ideal aspect ratio should be square or near-square for consistent display
- White version should be pure white (#FFFFFF) for maximum contrast on dark backgrounds
